## 背景
原 `BillingService` 使用 mock 数据，导致钱包页面无法反映真实余额和账单信息。

## 变更内容
- 重写 `src/main/java/com/example/tengxuncvm/service/BillingService.java`
  - 使用 TencentCloud `CommonClient` 直连 Billing OpenAPI（`billing` / `2018-07-09`）
  - 余额：调用 `DescribeAccountBalance`
  - 账单：优先调用 `DescribeBillSummaryByProduct`，失败后降级 `DescribeBillSummaryByPayMode`
  - 增加多字段兼容解析，适配不同返回结构

## 影响范围
后端账单接口：
- `GET /api/billing/balance`
- `GET /api/billing/bills`
- `GET /api/billing/bills/{billId}`

## 验证方式
- `mvn -q -DskipTests package` 通过
- 服务已重启，可在钱包页验证真实余额/账单是否返回

## 注意
- 账号需具备 Billing 相关 API 权限，否则接口会报鉴权错误。
